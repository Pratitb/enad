Project Name - SALES OFFICE, Suraksha Smart City

Location - Vasai East

Status - Completed

Developed Area - 11,840 sq.ft

Description - The sales office is designed for the prospective customers. The idea is to create a lasting first impression among the users coming in. The entrance was designed to give a sense of grandeur yet at the same time gradually transits the user from exterior to the interior. The threshold creates space as an condition to allow the experience to gradually set in the users.

Project Name - PODIUM GARDENS, Suraksha Smart City

Location - Vasai East

Status - Ongoing

Developed Area - 18,115 sq.ft

Description - 4 R.G. areas located upon Podiums dedicated to four activities i.e. kids play area , meditation and exercise ,sports turf and nana-nani park respectively. Each R.G. is designed to cater the needs of particular age groups and allow uninterrupted programs.

Project Name - SPORTS CLUB, Suraksha Smart City

Location - Vasai East

Status - Ongoing

Developed Area - 18,113 sq.m.

Description - 3 different type of sports facility were provided in the layout.
